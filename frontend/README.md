# AVERRA 🌍  
### AI that watches over Earth

AVERRA is an AI-powered disaster intelligence platform that uses satellite imagery and deep learning to detect natural disasters, assess damage severity, and support faster disaster response.

## 🚀 Features
- Disaster detection using satellite images
- Damage severity classification
- Visual segmentation of affected zones
- Real-time alerts and analytics dashboard
- Government-ready reports

## 🧠 Tech Stack
- AI: PyTorch, OpenCV, Scikit-learn
- Backend: FastAPI
- Frontend: React, Tailwind CSS
- Maps: Leaflet / Mapbox
- Deployment: Vercel, Render
