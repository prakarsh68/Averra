# 🚨 Averra

AI-Powered Disaster Intelligence & Emergency Response Platform

## Overview

Averra is a real-time disaster monitoring and response dashboard designed to improve situational awareness during emergencies. The platform combines live incident reporting, AI-driven risk forecasting, operational intelligence, and resource tracking into a single command center interface.

## Features

### 📡 Real-Time Incident Monitoring
- Live disaster reports from users
- Firestore real-time synchronization
- Incident categorization and tracking

### 🤖 AI Prediction Engine
- Disaster probability forecasting
- Regional risk assessment
- Multi-hazard analysis
- 72-hour prediction window

### ⚠️ Threat Intelligence
- Dynamic threat level monitoring
- Active alert tracking
- Risk score calculation
- Mission status overview

### 🗺️ Geospatial Intelligence
- Interactive disaster map
- Location-based incident visualization
- Disaster hotspot monitoring

### 📊 Command Dashboard
- Active alerts monitoring
- Response coordination
- Resource tracking
- Operational analytics

## Installation

Clone the repository:

```bash
git clone https://github.com/prakarsh68/Averra.git
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
